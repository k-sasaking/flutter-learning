# 課題1 プロフィールアプリを作りなさい。

以下の条件に合ったアプリを作成しなさい。

1. 以下の情報を記載する

    - 名前
    - コース
    - 部活/委員会
    - 好きな〇〇

1.  テーマの色を変更する（blue以外を選択すること。）
1.  アイコンをハートマーク（favorite）に設定する。
