# 条件分岐

条件分岐とは、例えばスイッチが ON の時と OFF の時で違う処理をするときに使います。

条件分岐は、if 文を使います。

## if 文の使い方

以下のプログラムを実行してみましょう。

```
void main() {
  int age = 15;
  print("年齢を確認します。");

  if(age == 15){
     print("あなたは15才です。");
  }
}

```

想定結果

```
年齢を確認します。
あなたは15才です。
```

続いて、age の値を 20 に変えて実行してみましょう。

```
void main() {
  int age = 20;
  print("年齢を確認します。");

  if(age == 15){
     print("あなたは15才です。");
  }
}

```

想定結果

```
年齢を確認します。
```

このように、age の値が 15 だと表示され、それ以外の場合だと表示されないような処理を条件分岐と言います。

条件分岐をする際は、以下の if 文を使います。

```
if(　条件式　){

    条件式が成り立った時の処理

}

```

### ※ インテンドを揃える

プログラムで{}が出てきた時は、その中の行は２文字分の半角スペースを空けましょう！

この開けるスペースのことを**インテンド**と言います。

インテンンドを揃えると、プログラムのコードがきれいに見えます。

（悪い例）

```
void main() {
int age = 20;
    print("年齢を確認します。");
  if(age == 15){
print("あなたは15才です。");
}
if(age == 20){
print("あなたは20才です。");
  }
}
```

（良い例）

```
void main() {
  int age = 20;
  print("年齢を確認します。");

  if(age == 15){
     print("あなたは15才です。");
  }
  if(age == 20){
     print("あなたは20才です。");
  }
}
```

## 条件式

条件式は、その条件が ○ か × かを判断する式です。

○ の時を true,× の時を false といいます。

<tr><td>a&lt;=b</td><td>aがb以下の時true</td></tr>

### bool 型

この条件式の結果の true, false を bool 型と言います。

変数に入れることも可能です。

```
void main() {
  int age = 20;
  bool isOk = true;

  print("年齢を確認します。");
  print(isOk);
}
```

想定結果

```
年齢を確認します。
true
```

## if-else 文の使い方

if 文は条件式が true の時に処理をしました。

条件式が false の時に処理をする場合は、else 文を使います。

以下のプログラムを実行してみましょう。

```
void main() {

  int age = 30;


  if(age < 18){
    print("あなたは子供です");
  }
  else {
    print("あなたは大人です");
  }
}
```

上記のプログラムで、age の値を 10 にして実行してみましょう。

このように、if の条件以外の場合、処理を行います。

```
if (/* 条件式 */) {

 /* trueの時の処理 */
}
else {
 /* falseの時の処理 */
}

```

## 3 つ以上の条件分岐

if-else 文だと、２つにしか分岐できませんが、３つ以上に条件を分岐させたいときは、else if 文を使って表現をすることができます。

以下のプログラムを実行してみましょう。

```
void main() {

  int age = 70;

  if(age < 18){
    print("あなたは子供です");
  }
  else if(age > 65){
    print("あなたはシニアです");
  }
  else {
    print("あなたは大人です");
  }
}
```

想定結果

```
あなたはシニアです
```

age の値を変えて、ちゃんと条件分岐しているか試してみましょう。

このように、if の条件以外の場合、処理を行います。

```
if (/* 条件式1 */) {

 /* 条件1がtrueの時の処理1 */
}
else if (/* 条件式2 */) {
  /* 条件2がtrueの時の処理2 */
}
else {
 /* falseの時の処理 */
}

```

この場合の処理の流れは以下のようになります。

- 条件 1 に当てはまるかどうか確認
- 条件 1 に当てはまるなら処理 1 を実行。
- 条件 1 に当てはまらない場合、条件 2 に当てはまるかどうか確認
- 条件 2 に当てはまるなら処理 2 を実行。
- 条件 2 に当てはまらない場合、else を実行
